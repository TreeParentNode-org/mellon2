# Mellon - Your Personal Knowledge Workspace

> "Speak, friend, and enter."

Mellon is a minimalist text editor and workflow tool combining Zettelkasten linking, AI-powered LaTeX typesetting, and git-style version control.

## Features

- **Zettelkasten Linking**: Connect ideas with [[wiki-style links]]
- **AI LaTeX Interpretation**: Natural language â†’ LaTeX via Claude AI
- **Git-Style Versioning**: Commit and restore workspace snapshots
- **Minimalist Design**: Distraction-free interface
- **Auto-Save**: Never lose your work

## Quick Start

```bash
# Clone the repository
git clone https://github.com/yourusername/mellon.git
cd mellon

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide Icons
- Claude AI API
